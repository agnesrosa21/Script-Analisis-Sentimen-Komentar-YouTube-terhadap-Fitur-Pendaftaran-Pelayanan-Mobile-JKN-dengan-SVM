# Script-Analisis-Sentimen-Komentar-YouTube-terhadap-Fitur-Pendaftaran-Pelayanan-Mobile-JKN-dengan-SVM
Repository ini berisi script analisis sentimen komentar YouTube terhadap fitur pendaftaran pelayanan Mobile JKN menggunakan Support Vector Machine (SVM). Input yang digunakan meliputi commentsdata_svm.csv (data komentar), kamus_normalisasi.csv, dan stopwords-id.txt.
